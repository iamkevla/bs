G'day!

Twitter Bootstrap components executed as AngularJs Directives to provide a rich new DSL (Domain Specific Language). The aim here is to demonstrate the power and the simplicity of Angular Directives.

Dropdown menus

Example

Toggleable, contextual menu for displaying lists of links.

    model.menuItems = [
        { menuItem: "Action", href:"#" },
        { menuItem: "Another action", href:"#" },
        { menuItem: "Something else here", href:"http:www.krackas.com" },
        { type: "divider"},
        { menuItem: "Separated link", href:"#" }
      ];

<bs-dropdown-menu menu-items="model.menuItems" trigger="Dropdown Trigger" />

Button groups

Examples

Two basic options, along with two more specific variations.

Single button group

Wrap a series of buttons with bs-btn in bs-btn-group.

Left Middle Right 
<bs-btn-group>
  <bs-btn>Left</bs-btn>
  <bs-btn>Middle</bs-btn>
  <bs-btn>Right</bs-btn>
</bs-btn-group>

Multiple button groups

Combine sets of <div class="btn-group"> into a <div class="btn-toolbar"> for more complex components.

1 2 3 4 5 6 7 8 
<bs-btn-toolbar>
  <bs-btn-group>
    ...
  </bs-btn-group>
</bs-btn-toolbar>

Vertical button groups

Make a set of buttons appear vertically stacked rather than horizontally.

<bs-btn-group btn-group-vertical>
  ...
</bs-btn-group>

Progress bars For loading, redirecting, or action status

Examples and markup

Basic

Default progress bar with a vertical gradient.

<bs-progress value="{{model.progressMeter}}"  />

Striped

Uses a gradient to create a striped effect. Not available in IE7-8.

<bs-progress value={{model.progressMeter}} progress-striped />

Animated

Add active to progress-striped to animate the stripes right to left. Not available in all versions of IE.

<bs-progress value={{model.progressMeter}} progress-striped active />
